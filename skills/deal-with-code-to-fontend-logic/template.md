### 列表与分页

- `list`: 表格数据源，每项为 `record`
- `total` / `pageSize` / `current`: 分页数据
- `loading`: 表格 loading
- `onPageChange(page)`: 翻页回调
判断：
这种过于细节，保留一页多少行、是否分页，能不能切换一页几行即可
